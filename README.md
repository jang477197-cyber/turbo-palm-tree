
과제1

# 1. 분수 추상 자료형(ADT) 정의

본 과제에서는 정수형 분자와 분모로 이루어진 분수 자료형을 추상 자료형(Abstract Data Type, ADT)으로 정의한다.
분수는 하나의 자료형으로 취급되며, 내부적으로 분자와 분모를 가지는 구조체로 구현된다.

---

# 2. 자료 구조

분수는 다음과 같은 형태로 정의된다.

* 분자 (numerator): 정수형 값
* 분모 (denominator): 정수형 값 (0이 될 수 없음)

이를 통해 분수는 num/den 형태로 표현된다.

---

# 3. 연산 정의

본 분수 ADT는 다음과 같은 연산을 제공한다.

1. 덧셈(add)
   두 분수 a/b와 c/d에 대해 다음과 같이 계산한다.
   (a/b) + (c/d) = (ad + bc) / bd

2. 뺄셈(sub)
   (a/b) - (c/d) = (ad - bc) / bd

3. 곱셈(mul)
   (a/b) × (c/d) = (ac) / (bd)

4. 나눗셈(div)
   (a/b) ÷ (c/d) = (ad) / (bc)

---

# 4. 기약분수 처리

모든 연산 결과는 최대공약수(GCD)를 이용하여 기약분수 형태로 변환한다.
이를 통해 결과는 항상 가장 단순한 형태로 유지된다.

---

# 5. 예외 처리

* 분모가 0인 경우 오류로 처리한다.
* 나눗셈에서 0으로 나누는 경우 오류를 발생시킨다.

---

# 6. 입출력 형식

입력은 다음과 같은 형태로 주어진다.

```id="7yl6j5"
2/5 + 2/4
```

출력은 연산 결과를 기약분수 형태로 출력한다.

```id="y3z72c"
9/10
```

---

# 7. 결론

본 과제를 통해 분수를 하나의 추상 자료형으로 정의하고,
해당 자료형에 대한 연산을 함수로 구현함으로써 자료의 추상화 개념을 이해할 수 있다.

과제2

# 1. 최대공약수와 최소공배수

본 과제에서는 두 정수에 대한 최대공약수(GCD)와 최소공배수(LCM)를 계산하는 프로그램을 구현한다.

---

# 2. 최대공약수(GCD)

최대공약수는 두 정수를 모두 나눌 수 있는 가장 큰 정수를 의미한다.
본 프로그램에서는 유클리드 호제법을 이용하여 이를 계산한다.

유클리드 호제법은 다음과 같은 과정을 따른다.

* a를 b로 나눈 나머지를 구한다.
* b를 새로운 a로, 나머지를 새로운 b로 설정한다.
* 이 과정을 b가 0이 될 때까지 반복한다.
* 마지막 a가 최대공약수가 된다.

---

# 3. 최소공배수(LCM)

최소공배수는 두 정수의 공통 배수 중 가장 작은 값을 의미한다.

다음과 같은 공식을 이용하여 계산한다.

LCM(a, b) = (a × b) / GCD(a, b)

---

# 4. 입출력 예시

입력:

```
18 24
```

출력:

```
GCD: 6
LCM: 72
```

---

# 5. 결론

본 프로그램은 최대공약수와 최소공배수를 함수로 분리하여 구현함으로써 코드의 재사용성과 구조적 설계를 향상시켰다.

과제3

# 직사각형 둘레 및 넓이 계산 프로그램 (분수 확장)

## 1. 개요

본 프로그램은 명령줄 인자를 이용하여 분수 형태의 가로와 세로를 입력받아
직사각형의 둘레와 넓이를 계산하는 프로그램이다.

---

## 2. 입력 방식

프로그램은 다음과 같이 실행한다.

```
program.exe 3/7 4/9
```

* 첫 번째 인자: 가로 (분수)
* 두 번째 인자: 세로 (분수)

---

## 3. 계산 방법

* 둘레: 2 × (가로 + 세로)
* 넓이: 가로 × 세로

분수 연산은 다음과 같이 수행된다.

* (a/b) + (c/d) = (ad + bc) / bd
* (a/b) × (c/d) = ac / bd

---

## 4. 기약분수 처리

최대공약수(GCD)를 이용하여 결과를 기약분수 형태로 출력한다.

---

## 5. 실행 결과

입력:

```
3/7 4/9
```

출력:

```
Perimeter: 110/63, Area: 4/21
```

---

## 6. 결론

명령줄 인자를 이용하여 입력을 처리하고, 분수 연산을 통해
직사각형의 둘레와 넓이를 계산하였다.
